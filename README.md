# Aurget

A safe and simple, pacman-like interface to the AUR.

## Installation

Study the Arch [wiki][], then manually build and install [aurget][].

[wiki]:   https://wiki.archlinux.org/index.php/AUR
[aurget]: https://aur.archlinux.org/packages/aurget/

## Usage

~~~
$ aurget --help
$ man 1 aurget
$ man 5 aurgetrc
~~~
